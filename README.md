# Expenz
A full-stack expense tracking web application that helps users manage their income and expenses efficiently with real-time visual insights.

## Features
- User authentication using JWT
- Add, edit, and delete income & expense transactions
- Category-wise expense management
- Interactive charts for income vs expense analysis
- Secure and personalized user dashboard
- Responsive and clean user interface

## Tech Stack
**Frontend:** React, JavaScript (ES6), Chart.js, HTML, CSS  
**Backend:** Node.js, Express.js  
**Database:** MongoDB  
**Authentication:** JWT  

## Description
Expenz is designed to simplify personal finance management. The application uses React functional components and hooks for efficient state management and reusability. JWT-based authentication ensures data security, while Chart.js provides intuitive visualizations to help users better understand their spending patterns.

## Installation & Setup
1. Clone the repository  
   ```bash
   git clone https://github.com/your-username/expenz.git
2. Install dependencies
   npm install
3. Configure environment variables for MongoDB and JWT
4. Run the application
   npm start
